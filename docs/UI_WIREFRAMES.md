- Sidebar: 200px (collapsible to icon bar)
- Right panel: 250px (collapsible)
- Main content: Remaining space (min 600px)
- Bottom panel: 150px (collapsible)

- All panels resizable via drag handles
- Splitter positions saved in user preferences
- Minimum widths enforced to prevent unusable layouts

## 16. Animation & Transitions

### Watch Mode Animations
- **Power flow**: 300ms fade-in for energized elements
- **Blinking outputs**: 500ms on/off cycle (configurable)

## 17. Accessibility

### Keyboard Navigation
- Tab order: Logical left-to-right, top-to-bottom
- All buttons accessible via Tab
- Escape closes dialogs
- Ctrl+Tab switches editor tabs

### Screen Reader Support
- ARIA labels on all interactive elements
- Announce runtime state changes
- Describe watch mode highlights

### Color Blindness
- Don't rely solely on color (use icons + text)
- High contrast mode option
- Patterns for TRUE/FALSE (not just green/red)
